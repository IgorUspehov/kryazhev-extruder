Управление частотным преобразователем через Modbus TCP из Linux

Автор: Игорь Кряжев, технолог-консультант

1. Введение

Протокол Modbus TCP позволяет Linux-компьютеру управлять скоростью шнека экструдера, считывать ток двигателя и температуру ЧП по одному Ethernet-кабелю.

2. Регистры управления (на примере Delta VFD-EL)

  • Регистр 8192 (2000H): Команда Пуск/Стоп.
  • Регистр 8193 (2001H): Задание частоты (масштаб 0.01 Гц).

3. ПИД-регулирование

Для поддержания 500 кг/ч используется ПИД-алгоритм на Python, корректирующий частоту VFD в реальном времени.

# Коэффициенты ПИД для экструдера
Kp=0.15, Ki=0.02, Kd=0.05
