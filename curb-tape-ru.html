Лента бардюрная для клумб: выбор сырья и технология экструзии

Дата: 24 апреля 2024 | Автор: Игорь Кряжев

1. Виды бардюрной ленты и сырье

  • Из ПНД (HDPE): жесткая, держит форму. MFI 0,8-1,5. Цвет черный (с сажей). Цена низкая. Используется для разметки больших клумб.
  • Из ПП (полипропилен): более хрупкая, особенно на морозе. Дешевле HDPE. MFI 2-4. Для временной разметки.
  • Из ПВД (LDPE): мягкая, гнется, не держит форму. Используется для криволинейных рисунков. Смесь 70% LDPE+30% LLDPE.

Вторичное сырье: рециклат ПНД из канистр и флаконов (700-850 €/т) - идеален. PP из биг-бэгов (600-800 €/т) - дешевле, но качество хуже.

2. Технология экструзии ленты

Плоскощелевая головка (ширина 50-200 мм, толщина 2-5 мм). Температура: 190-210°C для HDPE. Вытяжка 1:2-1:3 для повышения прочности. Охлаждение в воде (10-20°C) - профилированные калибры. Намотка на гильзы (длина 25-50 м).

3. AI-контроль толщины

Я использую лазерный сканер (Linux Mint + Python) для непрерывного замера толщины ленты. Обратная связь с вытяжными валами (скорость корректируется). Точность поддержания толщины ±0,05 мм.
