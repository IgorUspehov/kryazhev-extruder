Einstellung des PID-Reglers für Extruder auf Python

Autor: Ihor Kryazhev, Technologieberater

1. Einführung

Ein Extruder ist ein System mit Totzeit. Die PID-Regelung (Proportional-Integral-Derivativ) ist notwendig, um Schwankungen der Ausstoßleistung auszugleichen, die durch Transportverzögerungen (30-120 Sek.) entstehen.

2. Koeffizienten (Ziegler-Nichols-Methode)

Für LLDPE-Recyclinglinien (500 kg/h) sind typische Werte:

  • Kp: 0,1 - 0,2 (Hz pro kg/h Fehler)
  • Ki: 0,005 - 0,015
  • Kd: 0,2 - 0,8

3. Anti-Windup und Rauschfilterung

Um eine Sättigung des Integrators zu verhindern, wird Anti-Windup implementiert. Messrauschen der Waage wird durch einen EWMA-Filter (Exponentially Weighted Moving Average) geglättet.

# Python EWMA Filter
filtered = alpha * measured + (1 - alpha) * last_value
