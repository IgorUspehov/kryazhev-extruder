Промышленная автоматизация: Управление частотным преобразователем по протоколу Modbus TCP на Python

Автор: Игорь Кряжев, технолог-консультант

1. Преимущества цифрового управления

Во вторичной переработке полимеров стабильность оборотов — залог качества. В отличие от аналогового управления (0-10В), Modbus TCP исключает влияние электромагнитных помех от нагревателей. Мы получаем полный контроль: задание частоты, чтение тока, момента и кодов ошибок по одному кабелю Ethernet.

2. Настройка сетевых параметров ЧП

Перед запуском скрипта на инверторе (Delta, ABB, Danfoss) необходимо задать статический IP (например, 192.168.1.10) и разрешить управление через коммуникационный порт.

Карта регистров (стандарт):
  • 0x2000: Control Word (Bit 0-1: Run/Stop).
  • 0x2001: Frequency Command (0-5000 для 50Гц).
  • 0x2103: Output Current (Ток в амперах).

3. Программная реализация на Python

Используем библиотеку pyModbusTCP для прямого управления шнеком экструдера.

from pyModbusTCP.client import ModbusClient
import time

class ExtruderDrive:
    def __init__(self, ip):
        self.client = ModbusClient(host=ip, port=502, auto_open=True)

    def set_speed(self, hz):
        # Преобразование Гц в значение регистра
        val = int(hz * 100)
        if self.client.write_single_register(0x2001, val):
            print(f"Скорость {hz} Гц установлена")
        
    def read_load(self):
        # Чтение тока нагрузки
        regs = self.client.read_holding_registers(0x2103, 1)
        return regs[0] / 10.0 if regs else 0

drive = ExtruderDrive("192.168.1.10")
drive.set_speed(42.5)
print(f"Текущая нагрузка: {drive.read_load()} A")
