Extruder PID Controller Setup in Python

Author: Ihor Kryazhev, Process Consultant

1. The Problem of Dead Time

Extruders suffer from transport delay. A standard P-controller causes oscillations; a PID controller with proper Kd (Derivative) action is required to handle sudden changes in material bulk density (agglomerate).

2. Sampling Time (dt)

The optimal sampling period dt for an extruder is 2–5 seconds. If dt < 2s, the controller reacts to sensor noise. If dt > 10s, it fails to correct output drops in time.

3. Implementation

class PID:
    def compute(self, measured):
        error = self.setpoint - measured
        self._integral += error * self.dt
        # Anti-windup logic here
        derivative = (error - self._prev_error) / self.dt
        return Kp*error + Ki*self._integral + Kd*derivative
