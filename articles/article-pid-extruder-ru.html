Настройка ПИД-регулятора для экструдера на Python

Автор: Игорь Кряжев, технолог-консультант

1. Введение

Экструдер — объект с транспортным запаздыванием (30–120 сек). ПИД-регулятор необходим для стабилизации производительности при изменении насыпной плотности сырья.

2. Коэффициенты для вторичного LLDPE

  • Kp: 0,1–0,2 (Гц на кг/ч ошибки)
  • Ki: 0,005–0,015
  • Kd: 0,2–0,8

3. Период дискретизации

Оптимальный период dt = 3 секунды. Это позволяет фильтровать шумы весового дозатора и сохранять скорость реакции.

# Базовая формула в коде
output = Kp * error + Ki * integral + Kd * derivative
