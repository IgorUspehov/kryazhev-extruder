VFD-Steuerung über Modbus TCP unter Linux

Autor: Ihor Kryazhev, Technologieberater

1. Einführung

Die Automatisierung einer Extrusionslinie erfordert eine präzise Steuerung der Schneckendrehzahl. Modbus TCP ermöglicht es Linux-Systemen, Frequenzumrichter (VFD) direkt über Ethernet zu steuern.

Vorteile: Diagnose in Echtzeit (Stromstärke, Temperatur) und PID-Regelung der Ausstoßleistung (kg/h).

2. Modbus-Register (Beispiel Delta VFD)

  • Register 2000H: Steuerwort (Bit 0: Start/Stop).
  • Register 2001H: Frequenzvorgabe (Skalierung 0,01 Hz).

3. Python-Integration

from pyModbusTCP.client import ModbusClient
client = ModbusClient(host='192.168.1.100', port=502, auto_open=True)
client.write_single_register(8193, 3000) # Setzt 30,00 Hz
